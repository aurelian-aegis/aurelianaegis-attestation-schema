# Specification

This directory contains the human-readable specification and governance documents for the AurelianAegis attestation envelope.

- `SPECIFICATION.md` is the primary specification entry point.
- Supporting documents define versioning, governance, signing, chain integrity, integration guidance, migrations, support policy, and publication guidance.
- `usage/` contains informative implementation and operational guidance for different platform patterns.

The current specification lives in this directory. Frozen machine-readable release snapshots are published under `schema/releases/`, `examples/releases/`, `test-vectors/releases/`, and `registry/releases/`.
