# Support Policy

## Active Support Window

- Core major `v1`: active support for 18 months after latest `v1` release.
- Security fixes for `v1` may continue for up to 6 months after active support.
- Only one major core line is actively developed at a time unless explicitly announced.

## End-of-Support Behavior

For questions about the schema or implementation support, email [info@aurelianaegis.io](mailto:info@aurelianaegis.io) or open a GitHub issue.

When a core major reaches end of support:

- No new extension/profile compatibility guarantees are made.
- No non-critical bug fixes are accepted.
- Security advisories MAY publish mitigation guidance without patch releases.
- As this is the first public specification release line, no prior-version migration document is required.
