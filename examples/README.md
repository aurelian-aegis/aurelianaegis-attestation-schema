# Examples

This directory contains human-readable example envelopes for the current version of the specification.

- Files at the top level are the current working examples used by the documentation.
- `releases/v1.0.0/` contains the frozen example set for the `v1.0.0` release snapshot.

Examples are illustrative and may use placeholder signatures for readability. Use `test-vectors/` for cryptographically verifiable interoperability fixtures.
